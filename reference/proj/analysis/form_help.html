Analysis Form

This form can be used to generate a data cube of image or record counts between the start date and end date.

The Time Period allows you to drill down and rollup by specifying which time periods are aggregated. Selecting "All" aggregates all date values and removes that column.

The User Class specifies which type of user to use in the data cube. If "All Users" is select, then the resulting data does not display an per user information and instead all rows represent values for any user in the remaining dimensions.

The Count lets you choose what is being counted. You can choose either images or radiological records.

All values must be specified before click the "submit" button.
