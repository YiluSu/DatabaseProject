Profile Help

Updating Personal Information

Personal information can be updated by changing the text in any of the following fields:

  • First Name
  • Last Name
  • Address
  • Email
  • Phone Number

Once you're happy with the chnages, click the "Update" button at the button. Once the changes have been made the system will return you to the homepage.

Changing Your Password

If you want to change your password, enter the new password into the "New Password" field. If this field is blank, your password will remain the same.
