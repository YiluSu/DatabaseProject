Analysis Results

The analysis results are displayed in a tabular representation of the data cube generated from the query parameters which are also displayed.

Any parameters which appear blank and the result of select either "All" for the time period or "All Users" for the user class. The results in a reduction of the dimension of the data cube. The highest dimension cube will have three dimensions: test type, date, and user. The lowest dimension cube will have only one dimension: test type.
